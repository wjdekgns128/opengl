컴퓨터 그래픽스 숙제2 과제 리드미 파일

압축파일 목록:

컴그화목정다훈2_실행파일 폴더 : exe파일

컴그화목정다훈2_cpp 폴더 : cpp파일만 들어있는 폴더


실행방법 : exe 파일 (실행파일폴더) 만약안되면 (CPP폴더에있는 파일들 비쥬얼스튜디오에 추가 -> 실행) 

res파일도 같이 꼭 옮겨주셔야합니다. (cpp,h파일에 있는 폴더에같이)

*하기전 채점 하실때 대비해서 제작해놓은 맵 데이터 파일도 같이 첨부합니다.*

*실행후 2D 모드에서 실행창에서 L/l 입력 그 후 도스창(CMD)창에서 test 라고 입력하시면 테스트 제작맵이 로드됩니다.*
* 로드후 바로 P/p 키보드 입력시 3d모드로 전환하여 바로 롤러코스터를 확인하실수있습니다..* 
* 물론 직접 맵을 만들 후 S/s 키를 입력하여 맵을 저장 할 수 있습니다.


1. 간단한 전체적인 설명:

처음에 2차원 탑뷰에서는 모든 오브젝트(나무,아파트,터널(동굴?),제어점)을 설치 및 높이(Z축) 값을 조절할수있습니다
그 후 3D 모드에서는 선로를 따라 롤러코스터가 이동합니다.

2. 구체적인 설명 :

처음에 2D 탑뷰에서는 F1 키로 제어점 설치& 이동 모드 F2 키로 오브젝트 설치 & 이동 모드 F3 키로 높이 조절 모드로 전환할 수 있습니다.
2D에서 F1를 키를 입력시 빈 공간에다가 마우스 왼클릭시 제어점을 설치 할 수 있으면 처음 제어점을 SP라는 폰트의 글씨로 시작점이 표시됩니다.
그 후 빈 곳에서 계속적으로 왼 클릭시 제어점이 생성되면 마지막 제어점은 항상 시작점의 제어점과 이어진 곡선(주황색 선)이 그려집니다. 또한 마우스 왼 클릭으로
제어점을 클릭하게되면 제어점이 마우스를 따라 이동하며 다시 왼쪽버튼을 클릭시 그 자리로 좌표값이 갱신됩니다(편집기능)


또한 F2 입력시 오브젝트를 설치 할 수 있으면 F2 클릭후, 1번 키는 나무 (초록색 세모) 2번키는 아파트(네모난 색상랜덤값인 오브젝트) 3번키는 터널을 설치 
할 수 있습니다.그 후 제어점과 마찬가지로 마우스 왼쪽 버튼으로 생성 할 수 있고 마우스 왼쪽으로 오브젝트 클릭시 이동할 수 있습니다.
여기서 터널을 생성할 수 있는 갯수는 1개로 고정되면 오브젝트 설치 & 이동모드(F2키 입력 후 ) 3번(터널 설치키)을 클릭시 터널이 5도씩
회전됩니다.(교수님한테 질문한 결과 직접 설치 할 수 있어도 된다고하셨습니다, 터널 위치를 주황색선(선로)와 일치하게끔 수동으로해야합니다 ㅠ)
그 후 처음에 13개의 오브젝트가 생성되고 5개의 갈색 원(등대) 이 생기는데 이 등대는 설치 할 수 없고 이동만가능합니다.

F3클릭시 높이 조절 모드로 모든 오브젝트 (제어점포함 나무,등대,도시,터널)의 높이를 조절 할 수 있습니다. F3 클릭시 폰트로 각 높이 값이 나오고
마우스 오른쪽버튼으로 높이 감소(-10) 왼쪽버튼으로 높이증가(+10)을 할 수 있으면 최대 높이는 각 각 오브젝트마다 제한이 있습니다.

2D모드에서  S/s 키는 세이브, L/l는 로드키 입니다. S를 입력을 하면 도스창(CMD)에 맵 이름을 입력하면 파일이 저장되고 L/l키를 입력후 
그 파일 이름을 입력하면 로드가됩니다. 

그 후 p/P 키를 입력하면 3D 모드로 전환하게됩니다. 3D 모드에서는 F1 : FPS 카메라 모드 F2 : 일정 좌표에 고정되있는(마우스로 좌우만 )볼수있는 테스트
카메라 모드, F3 : 카메라 자유 시점 모드로 바뀌게되고 총 4개로 이루어진 롤러코스터가 선로에따라 이동하게됩니다. 맵은 스카이박스(텍스쳐매핑)을 이용하여
구현 하였습니다. 롤러코스트는 높이(z축)값에 따라 올라갈떄는 느리게 내려갈때는 빠르게 내려갑니다(가속도)
또 맵 맨 위쯤에 빨강색 원(태양) 푸른색 원(달) 이 존재하고 이 두개는 계속 움직이게하여 낮과밤의 느낌을 주고자하였습니다.
뿐만 아니라 낮과 밤이 있어 화면이 점점 어두워지고 점점 밝아집니다(조명) 화면이 일정 수준 어두워지면(밤) 
처음에 생성된 5개의 갈색 원(등대)가 360도 회전하면서 바라보고있는 부분만 밝힙니다(등대와 같이) 또한 스페이스바 키를 통해 폭죽(총알)을 발사
할 수있으면 맵밖을 벗어나려고하려나 일정시간이지나면 터지면서 파티클이일어납니다. 또한 처음에는 밝은 날씨였지만 시간이 지나면 눈,비가 옵니다
이때는 화면의 최대 밝기가 줄어들어 밤이 길어보이는 효과를 주고자 하였습니다.

3. 키 조작 설명
2D 모드일때  :

마우스 왼쪽 : F1 , F2 ,F3 각각 모드에 따라 제어점설치&이동,오브젝트설치&이동,높이 증가)(클릭으로)

마우스 오른쪽 : F3,높이조절모드에서 높이감소 (오브젝트를 클릭)

F1 : 제어점 설치 및 이동 가능 모드 

F2 : 오브젝트 설치 및 이동 가능 모드 
( 이때 숫자 1,2,3 키로 각각 나무(초록색세모),아파트(네모모양),터널( 큰 횐색네모),생성은 못하지만 이동 할 수 있는 등대 5개(갈색 모양의 원) 
터널은 한개만있는 상태에서 3번을 누르면 5도씩 회전.

F3 :높이 조절 모드 : 마우스 오,왼 버튼으로 높이 감소 증가 ( 폰트로 높이 표시)

L/l : 로드 모드입니다. 파일 이름을 도스창(cmd)창에 입력하면 그 파일을 읽습니다. (파일경로 /res/Map/ 

S/s : 현재 맵을 저장하는 모드입니다. 파일 이름을 도스창에 입력하면 그파일이 저장됩니다 (파일경로 /res/Map/) 

3D 모드일때:

F1 : FPS 형식(1인칭) 롤러코스트를 따라다니는 카메라 (마우스,키보드 움직임 불가)

F2 : 테스트 카메라 형식, 일정 위치에서 바라보는 카메라 ( 좌표 고정 , 키보드 움직임 불가, 마우스로 상하좌우 바로보기가능)

F3 : 3인칭 카메라 : WASD로 카메라이동 마우스로 상하좌우 바라보기가능

스페이스바 : 총알발사

N/n : 날씨를 강제로 변경(테스트) 맑은날씨(아무것도 안내림) - > 눈 - > 비 순으로 강제적으로바뀜. 입력안하실 시간이 지나면 자동으로 전환.

공통 모드 :

Q/q 종료

F2,F3 입력시 화면에 마우스가 고정됩니다. 다른 작업을 하시고싶을때는 f1 (fps모드로) 변경후 하시면됩니다!

4. 구현 기능
숙제 PDF 나와있는 제시해준 기능중  가능한 기능

처음에 실행시 임의의 위치에 임의의 갯수만큼(10개이상) 장애물 설정
2차원 화면에서 마우스로 편집,이동 가능
곡선의 제어점으로 편집
3차원 프런트뷰(원근 투영) 가능
롤러코스트 3개이상 열차 연결되어 선로따라이동 가능
터널 생성 가능
키보드 명령에 의해 총알발사 가능
시점 변환 가능
카메라 변환(이동,마우스로 회전)가능
날씨 변환 가능
속도 변환 가능(내려갈때는빠르고 위로갈떄는 느림)

--PDF 파일에서 제시하는 기능은 전부 가능.--

- 낮과 밤 개념추가 (배경 조명 추가)
- 모든 오브젝트 설치 이동 높이 조절가능
- 아파트 추가
- 등대 추가(스프트라이트) 
- 총알(폭죽) 터질때 파티클 추가
- 폰트로 높이 표시 추가
- 파일 입출력(맵로드,맵세이브)
